Detailed Description

Defines the contract for a timer service that provides periodic task execution functionality.

This interface extends IDisposable to ensure proper cleanup of timer resources. Implementations should handle timer initialization, execution, and cleanup.

Starts the timer with the specified interval and callback function.

Parameters
intervalThe time span between timer ticks.
callbackAn asynchronous function to be executed on each timer tick.

The callback function will be invoked periodically based on the specified interval. If the timer is already running, it should be stopped and restarted with the new parameters.
