Detailed Description

Converts boolean values to auto-refresh status text for UI display.

This converter is used in XAML bindings to display the current auto-refresh state as user-friendly text. It converts true/false values to appropriate action text indicating whether auto-refresh can be started or stopped. Implements IValueConverter for use in XAML data binding.
